## 设置验收

- [ ] 外观:跟随系统/浅色/深色切换后主窗口和设置窗口即时更新。
- [ ] 界面语言:系统/中文/English 切换后主要 UI 文案更新。
- [ ] 翻译引擎:本地翻译、LLM、Google、DeepL、有道可选择;非官方引擎显示不稳定提示。
- [ ] 凭据与测试连接:LLM API Key、有道 appKey/appSecret 只存 Keychain;测试连接成功和失败都有明确提示。

## 错误分支验收

- [ ] 文件不可读:选择无权限或损坏文件时提示无法读取。
- [ ] 非 PDF:翻译入口拒绝非 PDF 并提示格式错误。
- [ ] 加密 PDF 密码错误:提示密码错误并允许重试。
- [ ] 无文字可识别:空白/不可识别 PDF 报错清楚。
- [ ] 非中英文档:提示仅支持中英文档,可手动选择中文/英文方向重试。
- [ ] 本地翻译语言包缺失:提示下载系统语言包。
- [ ] 引擎 Key 无效:提示密钥无效,并建议切换本地翻译或其他引擎。
- [ ] 引擎限流:提示稍后重试,并建议切换本地翻译或其他引擎。
- [ ] 引擎不可用:提示引擎不可用,并建议切换本地翻译或其他引擎。
- [ ] 网络错误:提示网络错误,并建议切换本地翻译或其他引擎。
- [ ] 导出写入失败:提示保存失败,保留预览内容,允许重新选择保存位置。
- [ ] 用户取消:运行中点击取消后回到 idle,不显示误导性成功状态。

## 打包验收

- [ ] `make bundle` 成功生成 `dist/PDFlab.app`。
- [ ] `dist/PDFlab.app/Contents/Info.plist` 包含 `CFBundleIdentifier=com.pdflab.app` 和 `LSMinimumSystemVersion=15.0`。
- [ ] `codesign --verify --deep --strict dist/PDFlab.app` 通过。
- [ ] 双击 `dist/PDFlab.app` 可启动主窗口。
- [ ] 打包版可打开 PDF、执行一次翻译预览、保存导出文件、立即对照查看。
